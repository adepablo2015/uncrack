Explicación del código : Declara el tipo de documento, indicando que es un documento HTML5. : La etiqueta raíz que envuelve todo el contenido de la página. lang="es" especifica el idioma. : Contiene metadatos sobre la página. : Define la codificación de caracteres para mostrar correctamente los caracteres especiales. : Configura la página para que se vea bien en dispositivos móviles. : Define el título que aparece en la pestaña del navegador. <link rel="stylesheet" href="style.css">: Vincula un archivo CSS externo para el estilo. <body>: Contiene todo el contenido visible del sitio web. <header>: Define la sección de la cabecera, que a menudo contiene el logotipo y la navegación principal. <nav>: Define los enlaces de navegación. <ul> y <li>: Crean una lista desordenada con elementos de lista para los menús. <a>: Crea un enlace. href especifica la URL de destino. <main>: Representa el contenido principal del documento. <section>: Divide el contenido en secciones temáticas. <h1>, <h2>: Definen encabezados de diferentes niveles. <p>: Representa un párrafo de texto. <img>: Inserta una imagen. src indica la ruta del archivo y alt proporciona un texto alternativo. <footer>: Define el pie de página, que puede incluir información de contacto, derechos de autor, etc. <script src="script.js">: Vincula un archivo JavaScript externo para añadir interactividad.
